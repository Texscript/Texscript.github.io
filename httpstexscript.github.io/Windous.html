Soluciones generales.

  • Quitar hardware externo:

    Desconecta los dispositivos hardware que no sean imprescindibles. Entre estos podrían estar los siguientes: auriculares, impresoras, escáneres, altavoces, unidades flash USB y unidades de disco duro externas.

  • Actualizar Windows:

    Es aconsejable asegurarse de que todas las actualizaciones importantes estén instaladas antes de intentar actualizar Windows. Esto incluye las actualizaciones de los controladores de hardware del dispositivo.Desinstala cualquier software antivirus que no sea de Microsoft:Usa Windows Defender para proteger el dispositivo durante la actualización, ya que el software antivirus ajeno a Microsoft puede causar a veces problemas de actualización. Siempre que dispongas de medios de instalación y de toda la información requerida para la activación, siempre puedes reinstalar el software después de actualizarlo.

  • Desinstalar el software que no sea indispensable:

    El software obsoleto puede causar problemas con las actualizaciones de Windows, por lo que puede ser de ayuda quitar aplicaciones antiguas o que no sean imprescindibles.

  • Liberar espacio en disco:

    Para ver cuánto espacio de disco duro está disponible en el equipo, seleccione Explorador de archivos en la barra de tareas (conocido como explorador de Windows en Windows 7) o presione Windows tecla del logotipo + E.

Errores al actualizar:

  • No podemos completar las actualizaciones:

    Deshaciendo cambios, no apagues el equipo:Estos son ejemplos de errores genéricos que pueden aparecer cuando una actualización de Windows produce un error. Tendrás que determinar el código de error específico para investigar cómo poder solucionar el problema de la mejor manera posible.

  • Error al configurar las actualizaciones de Windows:

    Revirtiendo cambios:Puedes encontrar el código de error de la actualización que ha fallado en el historial de actualizaciones. Busque la actualización que no se instaló, anote el código de error y, a continuación, póngase en contacto con el soporte técnico de Microsoft.

  • La actualización no se puede aplicar a tu equipo:

    Este error puede indicar que tu PC no tiene las actualizaciones necesarias instaladas.Asegúrate de que todas las actualizaciones importantes estén instaladas en el PC antes de intentar actualizar.

  • Error: La actualización no se puede aplicar a tu equipo:

    La actualización no se puede aplicar a tu equipo: Este error puede indicar que tu PC no tiene las actualizaciones necesarias instaladas.Asegúrate de que todas las actualizaciones importantes estén instaladas en el PC antes de intentar actualizar.

Errores adicionales:

  • Errores 0xC1900101:

    Los errores que empiezan por 0xC1900101 suelen ser errores de controlador.Asegúrese de que el dispositivo disponga de suficiente espacio. Se necesitan al menos 16 GB de espacio libre en el dispositivo para actualizar un sistema operativo de 32 bits o 20 GB para un sistema operativo de 64 bits. Para obtener más información, vea Liberar espacio de unidad en Windows.

  • Error al configurar las actualizaciones de Windows:

    Revirtiendo cambios: Puedes encontrar el código de error de la actualización que ha fallado en el historial de actualizaciones. Busque la actualización que no se instaló, anote el código de error y, a continuación, póngase en contacto con el soporte técnico de Microsoft.
